Quick start

三步开始

1. 安装

python3 -m venv .venv
. .venv/bin/activate
python -m pip install -e .

2. 启动

python -m clawdone serve \
  --host 0.0.0.0 \
  --port 8787 \
  --token your-secret

3. 打开

http://<server-ip>:8787
